# Where AI Agents Fail

## Role in This Book
This English scaffold mirrors the Japanese opening chapter and fixes the same entry point: this book is about getting AI agents to complete work with artifacts and verification, not just to produce plausible output.

## Learning Objectives
- Explain the difference between a one-off correct answer and work completion
- Describe the four main failure modes of AI agents
- Understand the roles of the sample-repo, ChatGPT, and Codex CLI

## Outline
### 1. Getting one answer right is different from getting work done
### 2. Four failure modes: wrong answer, forgetting, breaking things, stopping early
### 3. Prompt Engineering, Context Engineering, and Harness Engineering
### 4. The recurring cases, sample-repo, ChatGPT, and Codex CLI
### 5. Read this book as a book about artifacts

## Exercises
1. Read a failed coding agent transcript and identify three missing artifacts.
2. Classify a weak task request by failure mode and propose a better direction.
