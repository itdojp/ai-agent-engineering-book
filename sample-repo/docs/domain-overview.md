# Domain Overview

support-hub は、社内サポートチームがチケットを扱うための最小ドメインモデル。

## Core Objects
- Ticket
- TicketStore
- SupportHubService

## Current Capabilities
- 一覧取得
- ステータス更新
- キーワード検索
- assignee での絞り込み

## Planned Expansion
- 監査ログの強化
- flaky test を題材にした verification chapter 用素材
- context pack と runbook の整備
