# English Templates Counterparts

This directory contains the English counterparts of reusable templates referenced by the manuscript. The canonical contract for all templates is defined in `templates/` (Japanese), and the files in `templates/en/` are derived English counterparts that are kept in sync with those canonical templates, not temporary placeholders.

## Current Scope

- Maintain English prompt-template counterparts for Appendix A
- Maintain English context-template counterparts for Appendix B
- Maintain English harness-template counterparts for Appendix C
- Keep the path stable for appendix and parity work
- Avoid mixing English template growth with Japanese template maintenance

## Out of Scope

- English translations beyond the prompt, context, and harness template sets
- Changes to the Japanese template contract
