# 図表一覧方針

このファイルは、ebook / print の後付けとして図一覧と表一覧を作るときの正本である。図そのものの source は `manuscript/figures/`、表そのものの source は各章の markdown table にある。

## 役割

- 読者が通読後に図と表だけを探し直せるようにする
- production で図番号、表番号、page を付ける前の seed を残す
- figure plan、章本文、組版物の対応を崩さない

## 図一覧の方針

- 図 ID、対象章、caption、first mention は `manuscript/figures/figure-plan.md` を正本にする
- 図一覧では「図から何を理解してほしいか」が 1 行で分かる caption を維持する
- 1 図 = 1 主メッセージを守り、組版後に図番号だけを差し替える

## 表一覧の方針

- 表一覧は reader-facing な表だけを載せる。箇条書きの代用品や一時比較表は載せない
- 表タイトルは、直前の小見出しと表の主メッセージを組にして production で確定する
- 現段階では page 番号を持てないため、「主に戻る章」と「近接見出し」を seed として残す
