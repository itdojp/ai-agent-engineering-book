## Goal
英語版原稿の scaffold と parity tracker を追加し、日本語版と対応関係を保ちながら英語版を育てられる状態にする。

## Reader Outcome
この issue の成果により、英語版原稿を日本語版と同じ章構成・brief 構造で管理し、後続の翻訳・編集作業を issue 単位で進められる状態にする。

## Inputs
- `AGENTS.md`
- 関連する local `AGENTS.md`
- `manuscript/`
- `manuscript/briefs/`
- `issue-drafts/manifest.json`
- `README.md`
- `scripts/verify-book.sh`

## Deliverables

- `manuscript-en/README.md`

## Acceptance Criteria
- `manuscript-en/` 配下に chapter / appendix の brief と skeleton が存在する
- chapter / appendix の file count が日本語版と一致する
- `manuscript-en/STATUS.md` で日本語版との対応関係と parity 状態を追跡できる
- `README.md` に英語版ディレクトリが反映されている
- `scripts/verify-book.sh` で英語版 scaffold の存在整合を確認できる
- verify が通る
- 残課題がある場合は明記する

## Out of Scope
- 各章の英語版を出版品質まで全文翻訳すること
- 英語版 sample-repo や英語版 docs / templates 本体の全面整備
- 日本語版本文の大規模な再編

## Verification
- `./scripts/verify-book.sh`
