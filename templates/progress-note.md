# Progress Note Template

## Status
- 任意。`in-progress`、`blocked`、`ready-for-handoff` など粗い状態を書く

## Current Goal
- 今回の work package を 1 文で書く

## Completed
- 完了した作業を列挙する

## Decided
- この session で確定した判断を書く

## Open Questions
- 未解決事項を列挙する

## Last Verify
- 直近の verify command と結果を書く

## Changed Files
- handoff や review で追うべき差分を書く

## Resume Steps
- 再開時に読む artifact と順序を書く

## Next Step
- 次の 1 手を 1 work package で書く
