# Context Pack Template

## Purpose
- State which task this context pack supports.

## Read Order
- List the order in which the worker should read the artifacts.

## Canonical Facts
- Record the facts this task must not override.

## Live Checks
- Record the verification state, `Progress Note`, and other current checks that must be refreshed each time.

## Exclusions
- State the topics intentionally left out of this task.

## Done Signals
- Record the conditions that indicate the task is complete.
